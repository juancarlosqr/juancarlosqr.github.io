PROJECT

IoT Digital Wallet Project

Technology Stack React JS Redux Material-UI Express JS Firebase

Mission - Create a simple concept to allow financial awareness on kids from 6 to 8 integrating the digital space with the real world in a seamless experience.

Description - The purpose of this project was to create an IoT experience that pairs a mobile app with an IoT-enabled object, allowing parents to teach their childrens about the concept of digital money and simple financial education. The collaboration I had in this project was to build a mobile-friendly web app that included the following features: display account balance, list transfers received, create a goal, list goals, show goal progress, list users and list IoT-enabled devices within the family.

Implementation - I was given the task to design and build the product both on the back-end and front-end, with completely freedom in terms of the technologies and frameworks I could use. I chose React and Redux as I think they are minimalistic but simple powerful libraries that would allow me to do fast prototyping and quick iterations, one of the project's requisites. Another factor was the recent knowledge gained at that time of those two libraries. On the back-end side I worked with Express JS and Firebase also for its simplicity and powerful features.

Cool aspects of this project - I was able to choose to work with React JS and Redux, as I really like them. I received all the application layouts and specifications from a designer and made the product a reality.
